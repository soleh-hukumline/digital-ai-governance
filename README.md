# Digital AI Governance Dashboard

This project is a high-performance Digital AI Governance Dashboard designed to provide a rigorous, data-driven forensic analysis of Indonesia's legal landscape regarding AI and cybersecurity. It maps the "vacuum of law" by integrating empirical cyber incident data with international and national regulatory frameworks using Legal Network Analysis (LNA).

## Features

- **Legal Network Analysis (LNA)**: Maps interdependencies between international standards, national laws, and empirical incident data.
- **Empirical Validation**: Utilizes a dataset of cyber incidents to identify structural holes in regulatory enforcement.
- **Interactive UI**: A modern, single-page application with a premium design for rigorous data exploration.

## Directory Structure

- `system/`: Python backend and analytical scripts (NLP, Legal Network Framework).
- `data/`: Datasets, including JSON files of cybersecurity incidents.
- `app/` & `index.html`: Web dashboard for visualizing the network analysis and empirical findings.
